# AGENTS — STP-Diff Research

## Project Overview
Research framework for discovering regulatory vulnerabilities in gene circuits. Combines Semi-Tensor Products (STP) with differentiable VJP operators for adversarial exploration of Boolean biological networks.

## Key Files
- `draft.md` — paper draft (primary writing file)
- `cited.md` — draft with inline citations
- `papers/stp-diff-paper.tex` — LaTeX source
- `papers/stp-diff-paper.md` — markdown export
- `papers/stp-diff-grn-vulnerability.md` — paper export
- `scripts/` — all Python experiment scripts (seeded: torch.manual_seed(42))
- `outputs/` — generated figures (PNG)
- `STP-GNN/` — PyTorch code repo (has its own git, github.com/yaskhalil/STP-GNN)

## Writing Conventions
- **Math formatting:** KaTeX (`$$...$$`) for equations
- **Sections:** `## Section N: Title`
- **Citations:** `[N]` format, references tracked in cited.md
- **Drafts:** Iterations saved to `drafts/` directory
- Keep the paper self-contained in `draft.md` — figures inline-referenced from `outputs/`

## Code Conventions
- Python scripts in `scripts/` — PyTorch-based
- Core engine: `STP_Diff_Research.py`
- Implicit STP operator via `torch.autograd.Function`
- Temperature-scaled softmax (τ=10.0) for differentiable Boolean logic
- Validation against DepMap data

## Constraints
- Do NOT modify `STP-GNN/` files without cross-referencing with the README there
- The STP-GNN subdirectory is a separate git repo (`github.com/yaskhalil/STP-GNN.git`)
- Figures in `outputs/` are generated by scripts in `scripts/` — regenerate, don't edit PNGs directly
- Paper math uses KaTeX — ensure `$$`, `\begin{bmatrix}`, and other LaTeX renders correctly
- The `drafts/` directory tracks iteration history — don't delete old drafts
